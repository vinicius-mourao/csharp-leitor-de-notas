# 📘 Sistema de Registro de Notas em C#

Projeto simples desenvolvido em C# que permite registrar nomes de alunos e suas respectivas notas, diretamente pelo terminal.

## Funcionalidades

* Inserir a quantidade de alunos
* Registrar o nome de cada aluno
* Inserir notas com validação de entrada
* Exibir todas as informações no final
* Tratamento de erros para entradas inválidas

## Tecnologias utilizadas

* C#
* .NET (Console Application)

## Estrutura do projeto

* `Program.cs` → Contém toda a lógica do programa
* Métodos principais:

  * `LerNotas()` → Responsável por coletar dados do usuário
  * `MostrarNotas()` → Exibe os resultados no console

## Como executar

1. Clone o repositório:

   ```bash
   git clone https://github.com/seu-usuario/seu-repositorio.git
   ```

2. Acesse a pasta do projeto:

   ```bash
   cd seu-repositorio
   ```

3. Execute o projeto:

   ```bash
   dotnet run
   ```
